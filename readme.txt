A WooCommerce plugin for selling configurable PC products with global compatibility checking, real-time pricing, and interactive configuration interface.

== Description ==

W2F PC Configurator is a powerful WooCommerce extension that allows you to sell configurable computer products. Customers can build their own PC by selecting components, with real-time price updates and compatibility checking.

== Features ==

* Create configurable PC products with multiple component types (CPU, Motherboard, RAM, GPU, Storage, PSU, Case, etc.)
* Global compatibility rules system for ensuring component compatibility
* Default configuration with starting price that switches to component pricing when changed
* Real-time price calculation and updates
* Interactive frontend configuration interface
* Running spec summary
* Share configuration via URL
* Full WooCommerce cart and order integration

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/w2f-pc-configurator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Ensure WooCommerce is installed and activated
4. Create a new product and select "PC Configurator" as the product type
5. Configure components and set default configuration
6. Create global compatibility rules in WooCommerce > PC Compatibility

== Frequently Asked Questions ==

= Does this require WooCommerce? =

Yes, WooCommerce 8.0 or higher is required.

= How do I set up a default configuration? =

When editing a PC Configurator product, go to the "PC Components" tab. For each component, select a default product. Set the default price in the "Default Configuration" section.

= How do compatibility rules work? =

Go to WooCommerce > PC Compatibility to create global compatibility rules. Rules can be based on product matches, attribute matches, or custom spec matches.
